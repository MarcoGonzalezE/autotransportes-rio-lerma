===========================
Vehicle Maintenance System
===========================

This application allows you to manage the preventive and corrective maintenance programs for trucks

It handles full Maintenance workflow:

 Order => Activities => Spare Parts Control => Order Release

Managing:
=========
* Unit Maintenance Program with Cycles
* Mechanics Activities
* Spare Parts Control
